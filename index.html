Belajar Instalasi RouterOS Menggunakan NAT dan Menghubungkan ke Client 1 dan Client 2 ( Client Berbasis Windows 7 )

Perkenalan

Perkenalan nama saya Mochammad Alfajjar Syachputra. Saya bersekolah di SMK Negeri 1 Mojokerto dengan jurusan TJKT atau Teknik jaringan dan telekomunikasi. Saat ini saya kelas 11 . Untuk tugas kali ini adalah "Instalasi RouterOS menggunakan NAT dan menghubungkan ke client". Saya akan menjabarkan langkah konfigurasi tersebut di sini.

Tujuan

Agar siswa/I dapat dan bisa menyetting routerOS menggunakan NAT dan dapat dihubungkan langsung kwe client melalui Lan Segment dan berbasis windows 7.

Persiapan

  • Alat dan Bahan

      Unduh di sini

    2. ISO Mirotik

      Unduk di sini

    3. Aplikasi Winbox 64x bit

      Unduh di sini

    4. ISO Windows 7

  • Langkah - langkah konfigurasi

  • Kalian juga bisa melihat video agar lebih jelas di sini
    1. Pertama pahami topologi di bawah ini : 

      Keterangan :
          -IP host PC : 2.2.22.1/26

          -IP Gateway 2.2.22.2

          -IP ether1 : 2.2.22.3/26

          -IP ether2 : 2.2.22.65/26

          -IP ether3 : 2.2.22.129/26
    2. Pastikan Laptop/PC tersambung dengan internet

    3. Buka Aplikasi VMWare dan Install RouterOS Bisa ikuti panduan berikut

    4. Jangan lupa masukkan ISO Mikrotik

      Apabila kurang jelas lihat di sini


    5. Tambahkan 2 Network Adapter

      network adapter - add



      Network adapter - finsih


      Apabila kurang jelas lihat di sini dan di sini


    6. Pada network adapter 2 dan network adapter 3, ubah ke LAN Segment

      Apabila belum ada LAN Segment, tambahkan LAN Segment dengan cara :add - ok
      KET :
      - Adapter 2 --> LAN Segment 1
      - Adapter 3 --> LAN Segment 2


      Apabila kurang jelas lihat di sini

    7. Setelah itu tekan OK, dan Jalankn Virtual mesin RouterOS nya
    8. Install RouterOS dengan perintah "A" , "I" , "Y" 

      KET :
            A = Pilih semua file
            I = Install semua file
            Y = Lanjutkan instalasi

      Apabila kurang jelas lihat di sini

    9. Setelah itu jalankan sesuai gambar di bawah ini 

      Ket :
  - interface pr = melihat apakah benar 
      terdapat 3 interface
  - ubah nama intercafe agar mudah di ingat.

      Apabila kurang jelas lihat di sini

    10. Setelah itu beri IP dimasing - masing interface sesuai ketentuan pemberian IP di atas.

      Apabila kurang jelas lihat di sini

    11. Buka Aplikasi Winbox dan cari IP yang tersambung ( ether 1 )

      Pilih MAc Address


      Apabila kurang jelas lihat di sini

    12. Cek IP Address apakah sudah sesuai

      IP --> Address

      Apabila kurang jelas lihat di sini

    13. Beri IP DNS

      IP --> DNS


      Apabila kurang jelas lihat di sini



      Lihat IP DNS internet / laptop

      Apabila kurang jelas lihat di sini 



      beri IP DNS :
  - 8.8.8.8 ( DNS google )
  - 8.8.4.4 ( DNS google )
  - 192.168.190.167 ( DNS internet /laptop )
  jangan lupa centang menu
    Allow Remote Requests

      Apabila kurang jelas lihat di sini

    14. Atur Firewall nya

      IP --> Firewall --> NAT --> tanda ( + )


      Apabila kurang jelas lihat di sini

      pada menu Action pilih masquerade

      IP --> Hospot --> Hospot SetUp -->ether3-hospot --> next


      Apabila kurang jelas lihat di sini

      Edit --> Virtual Network Editor
